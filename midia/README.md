# Mídia local

Coloque aqui as imagens dos produtos.

Estrutura recomendada:

```text
midia/
  produtos/
    TROF-MDF-001/
      foto1.jpg
      foto2.jpg
      exemplo.jpg
    TROF-ACR-001/
      foto1.jpg
      foto2.jpg
    MED-001/
      foto1.jpg
      exemplo.jpg
```

Nos CSVs, use caminhos relativos como:

```text
midia/produtos/TROF-MDF-001/foto1.jpg
```

No chat local, a ação sugerida abrirá o arquivo em:

```text
/midia/produtos/TROF-MDF-001/foto1.jpg
```
